# **App Name**: RoyalSplit

## Core Features:

- Person and Group Management: Add new people and create groups to organize shared expenses within an in-memory data structure. Data resets on page refresh.
- Expense Input and Splitting: Enter expense details including title, amount, payer, and choose between equal, unequal, or percentage split options. Supports various calculation methods identical to Splitwise.
- Automatic Balance Calculation: Dynamically calculates and updates individual balances for all participants based on new expense entries using Splitwise's core logic.
- Debt Summary Visualization: Clearly displays who owes whom within a group, including an optimized simplified debt logic to minimize transactions.
- Settlement Mechanism: Initiate settlement between individuals or a group, marking debts as settled and adjusting balances accordingly, replicating Splitwise's settlement algorithm.
- Expense History Log: Maintain and display a chronological list of all expenses added, providing a clear overview of financial activity.

## Style Guidelines:

- The primary color is a rich royal blue (#3333CC), selected for its noble and commanding presence. It provides excellent contrast against the dark background, ideal for prominent text and interactive elements.
- The background features a deeply desaturated shade of the primary blue (#16161D). This choice creates a dark, elegant canvas that underscores the app's premium feel without distracting from content.
- A vibrant golden yellow (#F7D56E) serves as the accent color. This bright hue provides a luxurious pop, drawing attention to important actions, icons, and highlighted information while complementing the royal blue.
- Headlines and prominent text will use 'Belleza' (humanist sans-serif) for an artistic and sophisticated touch. Body text will utilize 'Alegreya' (humanist serif) to ensure readability and add to the elegant, contemporary feel.
- Sophisticated, minimalist line-art icons or subtle glyphs will be used throughout the interface. These icons will maintain the 'modern minimal' aesthetic, enhancing clarity without clutter.
- The layout features a smooth, card-based design with rounded corners and soft, diffused shadows, creating a tactile and modern aesthetic. Content will be presented with generous spacing for a premium feel, adaptable using fluid grids and flexbox for full mobile responsiveness across all devices.
- Subtle, fluid animations will enhance user interactions, such as adding an expense, updating balances, or navigating between sections. These gentle transitions will contribute to the app's smooth and polished user experience.
