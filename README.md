# RoyalSplit

RoyalSplit is a smart expense-sharing application inspired by Splitwise.  
It helps groups split expenses, track balances, and settle debts with minimal transactions.

## Features

- Equal and custom expense splitting
- Simplified debt settlement algorithm
- Settlement simulation before confirming expenses
- Subset participant support
- Expense balance visualization
- Mobile responsive design

## Tech Stack

- HTML
- CSS
- JavaScript
- Firebase Hosting

## How It Works

The application calculates net balances for each participant and uses a greedy algorithm to minimize the number of transactions required to settle debts.
